PRÉSENTATION DU MINI-PROJET

Pour se préparer au mieux et saisir les enjeux du projet final, nous avons effectué durant quelques séances un "mini-projet".

Voici les étapes principales réalisées:

  1. 1. Récupération d'URL contenues dans un fichier texte
  2. 2. Récupération du code HTTP (=réponse à la requête) de chacune de ces URL & de l'encodage des pages
  3. 3. Insertion des données récupérer dans un format tabulaire
  4. 4. Création d'un tableau html pour la visualisation des résultats à partir du fichier tabulaire créé
  5. 5. Création de deux pages HTML
